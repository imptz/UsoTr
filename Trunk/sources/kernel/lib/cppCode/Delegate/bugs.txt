Невозможно использовать функции принимающие параметры по ссылке. 

class C
{

};

class A
{
	void func(C& c)
	{
	}
};

ICallObject* cObj = createCallObject(&A::func, &a, c);

В результате будет ошибка при компиляции из-за неоднозначности типа параметра "c"
